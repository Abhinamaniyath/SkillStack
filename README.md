SKILLSTACK

A personal skill-building tracker to organize, track, and visualize your learning journey across courses, videos, articles, and certifications.

#Features

*Learning Goals

    -Add skill name
    -Select resource type (video, course, article)
    -Choose platform (Udemy, YouTube, Coursera, etc.)

*Progress Tracking

    -Update status: Not Started → In Progress → Completed
    -Track hours spent
    -Add personal notes
    -Rate difficulty

* Dashboard Insights

    -Overview of learning progress
    -Category-wise breakdown
    -Skill growth visualization
    -Analytics to help you stay consistent

#Tech Stack

Frontend: React + Tailwind CSS
